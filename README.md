## Ping something and display a pop-up when it responds (for Mac)

### Usage
```
pingeroo 8.8.8.8
```

or to run in background and keep terminal functional
```
pingeroo mothernode-r05-whiskey &
```

### Install
```
git clone git@github.com:spectermax-glitch/pingeroo.git
cd pingeroo
bash install.sh
cd ..
source ~/.zshrc
rm -rf pingeroo
```
